# gatsby-plugin-markdown-pages

A Gatsby plugin for turning markdown files into templated pages.

## Install

With Yarn:

```bash
yarn add gatsby-plugin-markdown-pages
```

Or with npm:

```bash
npm install --save gatsby-plugin-markdown-pages
```

## Usage

```javascript
// In your gatsby-config.js
plugins: [
  'gatsby-plugin-markdown-pages',
]
```
